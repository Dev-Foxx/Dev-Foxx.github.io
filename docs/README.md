# 홈

*Jekflix 템플릿* 문서에 오신 것을 환영합니다!

여기서 템플릿으로 웹사이트를 설정하고 실행하는 데 필요한 모든 정보를 찾을 수 있습니다:

- 로컬 [환경 설정](setup.md#environment) 및 템플릿 [설치](setup.md#installing-template) 방법
- 웹사이트 [커스터마이징](settings.md#settings) 방법, 테마 색상, 레이아웃, 제목 등을 변경하는 방법
- Netlify를 사용하여 웹사이트를 [배포](netlify-cms.md#deploy-with-netlify)하고 [Netlify CMS](netlify-cms.md#integrate-with-netlify-cms)를 사용하여 콘텐츠를 편집하는 방법

아래 메뉴를 원하는 대로 탐색하시고, 가장 중요한 것은 즐기는 것입니다! 😄
